Learn how to use CSS Grid!

On the Grid is a tool that helps developers learn how to use CSS's most powerful layout tool: grid. When I first learned about grid, I was taken back. I learned once again that vanilla is the way to go! Less bloat, less dependecies. As of March 2017, Grid is supported by all major browsers.

Learn the Redux Cycle!

The Redux Visualizer is a tool built for React Riot, a hackathon that requires the use of React. It's a simple tool that allows you to see what's actually happening in the Redux cycle, using the Giphy API.
