Selected Work

  • Groovie

    Full-stack Developer

    Groovie is a web application for My Morning Jacket mega-fans. Having seen the band 60+ times, this is my true passion project.

    The app allows fans to track the MMJ concerts they've attended and get metrics returned based on that history. It was built with modern technologies like Ruby on Rails, React and Redux. I also got the chance to work with data-visualization libraries to display metrics, which is my new favorite thing.

  • On the Grid

    Front-End Developer

    CSS Grid is the newest and most powerful layout tool that ships with CSS. I built this app to teach web developers how Grid works, because it should be everywhere.

    It was built with pure, sweet Javascript and CSS.

  • Bulletin

    Mobile Developer

    This app was a real treat to make, as it was my first foray into mobile. I built this app with a couple pals using React Native. I love React, so I was pining to work with their cross-device version.

    The app allows users to drop (digital) notes for friends at any location - parks, resturants - you name it. Your friend will get the note when they show up at the location.
